# Mayerfeld Consulting – Practicum CSS Project

A responsive multi-page website built as a **group HTML & CSS practicum project**.

This project was created to practice **semantic HTML structure, modern CSS styling, responsive layouts, and collaborative Git/GitHub workflow**.

## 🌐 Live Demo
🔗 https://mayerfeld-consulting-practicum-css.vercel.app

## 📌 About the Project
The website was developed as a team project focused on building a professional and modern business-style website.

It includes multiple pages and sections that simulate a real-world consulting company website experience.

### Included Pages
- **Home Page**
- **About Page**
- **Gallery**
- **Contact Page**
- **Upcoming Events**
- **Digital Menu**

## 🛠️ Tech Stack
- **HTML5**
- **CSS3**
- **Flexbox**
- **CSS Grid**
- **Responsive Design**
- **Git & GitHub**
- **Vercel Deployment**

## ✨ Features
- Clean and modern UI
- Responsive layout for different screen sizes
- Reusable header and footer structure
- Organized file structure
- Team collaboration using branches and pull requests
- Deployed live with Vercel
